# dotfiles
Configuration files for software I use.

![preview](preview.png)

## Install
You can install my dotfiles using `stow`
```sh
% git clone git@github.com:romeq/dotfiles.git
% stow -t $HOME configs
```

## Note
- I've installed [vim-plug](https://github.com/junegunn/vim-plug) to my neovim. 
  You need it also.
